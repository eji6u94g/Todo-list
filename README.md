# To do List

Make "Microsoft To Do" with Vue 3.

## Project setup

```
$ npm install
$ npm run serve
$ json-server --watch data/db.json
```

## Tools and skill used in To do List

1. CSS / SCSS
2. Vue 3
3. Vuex
4. axios: RESTful API
5. uuid
6. vue-toastification
7. json-server: mock API
